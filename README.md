# Jot

A simple daily journal to jot down your day.

**Live:** https://jot-teal.vercel.app/

## What it does

- Track what you did yesterday and what you plan to do today
- Nest items with Tab/Shift+Tab
- Copy to clipboard for Slack or Markdown

## Run locally

```bash
npm install
npm run dev
```

## Built with

React, TypeScript, Vite, Radix UI
